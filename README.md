# VortexTech Cybersecurity - Week 1: OWASP Top 10 Vulnerabilities Research

Research and documentation of 5 vulnerabilities from the current **OWASP Top 10:2025** list, completed for the Vortex Tech Cyber Security Internship (Task 1).

## What this is

This repository contains a research document that explains five of the most critical web application security risks. For each vulnerability it covers:

- What it is, in plain language
- How an attacker could exploit it
- A real world example where it caused harm
- At least one practical way developers can prevent or mitigate it

## Vulnerabilities covered

1. Broken Access Control (A01:2025)
2. Injection (A05:2025)
3. Cryptographic Failures (A04:2025)
4. Security Misconfiguration (A02:2025)
5. Authentication Failures (A07:2025)
